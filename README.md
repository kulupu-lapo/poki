# poki-lapo
This ia a work-in-progress library and monolingual corpus for Toki Pona. Our vision is to expand to include all types of media in Toki Pona, including books, poetry, music, comics, posts and more. Files are transcribed by volunteers into Markdown format with metadata as shown below:
